| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-03 | Tampering | migrator.rs user inserts | mitigate | Use parameterized sqlx queries for all INSERTs; never interpolate parsed values |
| T-10-04 | Information Disclosure | password.rs | mitigate | format_md5_prefix only formats the hash string; never log password hashes (log only usernames and IDs) |
| T-10-05 | Tampering | id_map.rs | accept | Migration mappings table is write-once per (entity_type, old_id); PK constraint prevents duplicates |
