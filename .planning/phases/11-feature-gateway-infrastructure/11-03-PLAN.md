<task id="1">
<title>Add feature-gateway service to docker-compose.yml</title>
<action>
Add a `feature-gateway` service to `docker-compose.yml` following the judge-worker pattern:

```yaml
  feature-gateway:
    build:
      context: ./backend
      dockerfile: feature-gateway/Dockerfile
    container_name: online-judge-feature-gateway
    restart: unless-stopped
    ports:
      - "3001:3001"
    environment:
      DATABASE_URL: postgresql://postgres:postgres@postgres:5432/online_judge
      FEATURE_GATEWAY_ENABLED: "true"
      GATEWAY_BIND_ADDRESS: 0.0.0.0:3001
      WORKER_SECRET: ${WORKER_SECRET:-default_worker_secret_change_me}
      RUST_LOG: feature_gateway=debug
    depends_on:
      postgres:
        condition: service_healthy
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:3001/health"]
      interval: 30s
      timeout: 10s
      retries: 3
      start_period: 20s
```

Also update the `api` service:
- Add `FEATURE_GATEWAY_URL: http://feature-gateway:3001` to environment
- Add `feature-gateway` to depends_on list
</action>
