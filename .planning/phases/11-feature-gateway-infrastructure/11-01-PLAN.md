    Create `FeatureGatewayService` struct with fields: `db_pool: PgPool`, `cache: Arc<DashMap<String, ResolvedFeature>>`, `enabled: bool` (from FEATURE_GATEWAY_ENABLED env var per D-11).
    `fn new(db_pool: PgPool) -> Self`: Read `FEATURE_GATEWAY_ENABLED` env var (default "true"), parse to bool, create DashMap. If env var is "false", `enabled=false`.
    `fn cache_key(slug: &str, scope: &str) -> String`: Returns `"{slug}:{scope}"` format.
    `pub async fn resolve(&self, slug: &str, campus_id: Option<i64>, grade_id: Option<i64>) -> ResolvedFeature`:
      - D-11: If `!self.enabled`, return `ResolvedFeature { enabled: false, source: SystemEmergencyOff }` immediately.
      - Build cache keys for most-specific scopes: class (skip -- no class_id per Pitfall #3), grade, campus, global.
      - Check cache for each scope level (grade first if grade_id present, then campus, then global). If hit, return cached.
      - DB query: `SELECT enabled, scope FROM feature_flags WHERE feature_slug = $1 AND ((scope = 'grade' AND scope_id = $2) OR (scope NOT IN ('grade','class') AND scope = 'campus' AND scope_id = $3) OR (scope = 'global' AND scope_id IS NULL)) ORDER BY CASE scope WHEN 'grade' THEN 1 WHEN 'campus' THEN 2 WHEN 'global' THEN 3 END LIMIT 1` with (slug, grade_id, campus_id).
      - If row found: create ResolvedFeature with source mapped from scope string, insert into cache, return.
      - If no row: query `SELECT default_enabled FROM feature_registry WHERE slug = $1`. If found, return ResolvedFeature with enabled=default_enabled, source=Default. If not found, return disabled with source=Default.
    `pub async fn resolve_for_class(&self, slug: &str, class_id: i64, campus_id: Option<i64>, grade_id: Option<i64>) -> ResolvedFeature`:
      - Same as resolve but checks class scope first: cache key "class:{id}", DB WHERE includes `(scope = 'class' AND scope_id = $1)`.
    `pub fn invalidate(&self, slug: &str, scope: &str)`: `self.cache.remove(&Self::cache_key(slug, scope))`.
    `pub async fn list_registry(&self) -> anyhow::Result<Vec<FeatureRegistryEntry>>`: `SELECT * FROM feature_registry ORDER BY slug`.
    `pub async fn list_flags(&self, slug: &str) -> anyhow::Result<Vec<FeatureFlagEntry>>`: `SELECT * FROM feature_flags WHERE feature_slug = $1 ORDER BY scope`.
    `pub async fn set_flag(&self, slug: &str, scope: &str, scope_id: Option<i64>, enabled: bool) -> anyhow::Result<()>`: `INSERT INTO feature_flags (feature_slug, scope, scope_id, enabled) VALUES ($1, $2, $3, $4) ON CONFLICT (feature_slug, scope, scope_id) DO UPDATE SET enabled = $4, updated_at = now()`. After successful write, call `self.invalidate(slug, scope)`.
    `pub async fn delete_flag(&self, slug: &str, scope: &str, scope_id: Option<i64>) -> anyhow::Result<()>`: `DELETE FROM feature_flags WHERE feature_slug = $1 AND scope = $2 AND scope_id IS NOT DISTINCT FROM $3`. After successful delete, call `self.invalidate(slug, scope)`.
