    **2. Middleware: `backend/api-infra/src/feature_gateway/middleware.rs`**

    `pub fn feature_gate(slug: &'static str) -> impl Fn(Request, Next) -> Pin<Box<dyn Future<Output = Result<Response, StatusCode>> + Send>> + Clone`:
    Follow exact same pattern as `require_permission` in authz.rs (closure returning pinned boxed future, Clone).
    - Inside async block: Extract `TenantContext` from `req.extensions().get::<TenantContext>()`. If missing, return `Err(StatusCode::UNAUTHORIZED)`.
    - Get `gateway` from request: extract via `req.extensions().get::<Arc<FeatureGatewayService>>()` -- BUT this requires the gateway to be inserted into extensions. Alternative: use `State<AppState>` via `from_fn_with_state`. Since the authz pattern uses bare `from_fn` with extensions, we need to also store gateway in extensions OR use a different approach.
    - DECISION (Claude's discretion per CONTEXT.md): Use `from_fn_with_state` pattern. The feature_gate middleware takes `State(state): State<AppState>` internally. This requires wrapping differently than authz. Use this pattern:
    ```rust
    pub fn feature_gate_layer(slug: &'static str, gateway: Arc<FeatureGatewayService>) -> axum::middleware::FromFnLayer<impl Fn(Request, Next) -> ..., AppState> {
        // Actually, simplest is to follow the closure pattern but capture Arc<FeatureGatewayService>
    }
    ```
    - FINAL APPROACH: Follow the exact authz.rs pattern but capture `Arc<FeatureGatewayService>` in the closure. The gateway Arc is passed at router construction time (same way permissions are passed). The middleware signature matches authz.rs exactly.
    ```rust
    pub fn feature_gate(
        slug: &'static str,
        gateway: Arc<FeatureGatewayService>,
    ) -> impl Fn(Request, Next) -> Pin<Box<dyn Future<Output = Result<Response, StatusCode>> + Send>> + Clone {
        move |req: Request, next: Next| {
            let gateway = gateway.clone();
            let slug = slug;
            Box::pin(async move {
                let tenant_ctx = req.extensions().get::<TenantContext>()
                    .ok_or(StatusCode::UNAUTHORIZED)?;
                let resolved = gateway.resolve(slug, tenant_ctx.campus_id, tenant_ctx.grade_id).await;
                if resolved.enabled {
                    Ok(next.run(req).await)
                } else {
                    Err(StatusCode::NOT_FOUND)  // D-08: 404, not 403
                }
            })
        }
    }
    ```
