## Spacing Scale

Declared values (multiples of 4, from tailwind.config.js + existing component patterns):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px (`gap-1`) | Icon gaps, inline padding |
| sm | 8px (`gap-2`) | Compact element spacing |
| md | 16px (`gap-4`, `p-4`) | Default element spacing |
| lg | 24px (`gap-6`, `p-6`) | Section padding |
| xl | 32px (`gap-8`, `p-8`) | Layout gaps |
| 2xl | 48px (`gap-12`) | Major section breaks |
| 3xl | 64px (`gap-16`) | Page-level spacing |

Exceptions:
- Skeleton pulse animation: `animate-pulse` with 2s cycle (default Tailwind) -- acceptable exception
- Table row padding: `px-6 py-12` used in empty states -- follows 12px inner padding convention for tables

## Typography

Declared from `tailwind.config.js` fontSize scale. 4 primary roles:

| Role | Size | Weight | Line Height | Tailwind Class |
|------|------|--------|-------------|----------------|
| Body | 14px (0.875rem) | 400 (regular) | 1.5 (default) | `text-base` |
| Label | 13px (0.8125rem) | 500 (medium) | 1.5 (default) | `text-sm` |
| Heading | 16px (1rem) | 500 (medium) | 1.2 (tight) | `text-lg` |
| Display | 24px (1.5rem) | 600 (semibold) | 1.2 (tight) | `text-2xl` |

Extended sizes available but not primary roles:
- `text-xs` (12px): badges, timestamps, metadata
- `text-xl` (18px): page titles
- `text-3xl` (32px): hero headings
- `text-4xl` (40px): landing page display

Font weights in use: 400 (body), 500 (headings/labels), 600 (display). No bold (700) used in the design system.

Letter spacing: `tracking-tight` (`-0.02em`) applied to headings.

## Color

Source: `index.css` CSS custom properties with oklch for modern browsers, hex fallbacks.

| Role | Value (Light) | Value (Dark) | Usage |
|------|---------------|--------------|-------|
| Dominant (60%) | `#f7f7f8` (`--background`) | `#08090a` | Page background, main surface |
| Secondary (30%) | `#ffffff` (`--card`) / `#f5f5f8` (`--sidebar`) | `#14151a` (`--card`) / `#08090a` (`--sidebar`) | Cards, sidebar, nav, elevated surfaces |
| Accent (10%) | `#5e6ad2` (`--primary`) | `#7170ff` (`--primary`) | See reserved-for list below |
| Destructive | `#e5393d` (`--destructive`) | `#e5454a` (`--destructive`) | Delete confirmations, error actions only |
| Muted | `#7f7f8c` (`--muted-foreground`) | `#8a8f98` | Secondary text, placeholders, empty states |

Accent (`--primary`) reserved for:
1. Primary CTA buttons (submit code, create contest, save settings)
2. Active navigation items in sidebar
3. Link text (`text-primary`)
4. Focus rings (`--ring`)
5. Checkbox/radio active states
6. Chart primary color (`--chart-1`)

Semantic status colors (OJ-specific, not counted in 60/30/10 split):
- `--status-accepted` (#3ecf8e): Accepted verdict, AC badges
- `--status-wrong` (#cf2d56): Wrong Answer verdict
- `--status-pending` (#0a72ef): Pending/judging status
- `--status-tle` (#ffb287): Time Limit Exceeded
- `--status-re` (#fa7faa): Runtime Error
- `--difficulty-easy` (#10b981): Easy problems
- `--difficulty-medium` (#F7A501): Medium problems
- `--difficulty-hard` (#ff5b4f): Hard problems

| Element | Copy |
|---------|------|
| Primary CTA | Varies per page -- must be verb + noun: "жҸҗдәӨд»Јз Ғ" (Submit Code), "еҲӣе»әз«һиөӣ" (Create Contest), "дҝқеӯҳи®ҫзҪ®" (Save Settings) |

| Error state heading | "{entity}еҠ иҪҪеӨұиҙҘ" -- e.g., "жҺ’иЎҢжҰңеҠ иҪҪеӨұиҙҘ", "зҸӯзә§еҠ иҪҪеӨұиҙҘ" |
