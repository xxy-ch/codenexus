Skyline Partition Optimization

Time Limit 1.0s
Memory Limit 256.0 MB
Difficulty Points 2400

Description

Given a set of N rectangular buildings forming a skyline, determine the minimum number of rectangular partitions required to completely cover the silhouette without overlap.

Each building is represented by its start position L_i, end position R_i, and height H_i. The skyline is the upper envelope of these rectangles. A partition is defined as a rectangle that fits entirely within the silhouette.

Constraints

  • 1 ≤ N ≤ 105
  • 0 ≤ L_i < R_i ≤ 109
  • 1 ≤ H_i ≤ 109

Example Test Case

Input: 
3
1 5 10
4 8 15
7 10 8
Output: 
4

Explanation: The skyline can be covered by 4 non-overlapping rectangles placed strategically at height transitions.
