Skyline Partition Optimization

Given a set of n rectangular buildings in a city, your task is to partition the resulting silhouette into the minimum number of disjoint rectangles that satisfy the structural integrity constraints.

An architectural scholar seeks to represent this skyline with editorial precision. Each building is represented by three integers: L, H, and R, where L and R are the left and right x-coordinates, and H is the height.

analytics Constraints

  • 1 <= n <= 10^5
  • 0 <= L < R <= 10^9
  • 1 <= H <= 10^9
  • Time Limit: 1.0s
  • Memory Limit: 256MB

data_array Example Cases

Input

3
2 10 9
3 15 7
12 8 18

Output

[2, 10, 3, 15, 7, 10, 9, 0, 12, 8, 18, 0]

The skyline represents the overall contour of three buildings at coordinates (2,10,9), (3,15,7), and (12,8,18).
