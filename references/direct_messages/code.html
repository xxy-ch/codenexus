I switched to a sliding window approach. Here is the optimization I used:

solution.py 
def max_sum(arr, k):
    n = len(arr)
    if n < k: return -1
    
    window_sum = sum(arr[:k])
    max_sum = window_sum
    
    for i in range(n - k):
        window_sum = window_sum - arr[i] + arr[i + k]
        max_sum = max(max_sum, window_sum)
        
    return max_sum
