# 可发布版本范围清单（2026-03-06）

## 1. 本次发布纳入范围

- 用户主链路
  - 登录/注册
  - 题目浏览
  - 提交与结果查看

- 社区核心
  - 讨论：列表、详情、回复、点赞
  - 博客：列表、详情、创建、编辑、评论、点赞

- 竞赛核心
  - 竞赛列表/详情
  - 报名与榜单页面

- 教师基础
  - 班级列表与统计字段
  - 竞赛创建向导

- 管理员基础
  - 用户管理、题目管理、举报管理
  - 判题设置与题面配置
  - 私信模块（会话列表、消息列表、发送消息）
  - 反作弊模块（扫描配置、发起扫描、报告列表与详情）

## 2. 本次范围控制（可降级开关）

- 私信模块（默认开启）
  - 开关：`VITE_ENABLE_DIRECT_MESSAGES`（默认开启，显式设置 `false` 可关闭）

- 反作弊模块（默认开启）
  - 开关：`VITE_ENABLE_PLAGIARISM`（默认开启，显式设置 `false` 可关闭）

## 3. 发布包边界说明

- 发布后必须可演示：
  - 登录 -> 做题 -> 提交 -> 结果
  - 讨论/博客至少一条创建与互动链路
  - 管理端测试用例配置链路

- 本次要求可演示：
  - 私信真实会话
  - 反作弊扫描任务与报告查看

## 4. 追补计划

- 若线上发现异常，可通过以下开关即时降级：
  - `VITE_ENABLE_DIRECT_MESSAGES=false`
  - `VITE_ENABLE_PLAGIARISM=false`
